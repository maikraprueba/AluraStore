# Análisis de Datos - Alura Store Latam

**Autor:** Maycol Ninaja

## Descripción
Este proyecto presenta un análisis exploratorio de datos de cuatro tiendas de Alura Store Latam utilizando Python. El objetivo es evaluar el desempeño de cada tienda mediante métricas clave y visualizaciones para obtener conclusiones basadas en datos.

El análisis se desarrolla en un notebook de Jupyter / Google Colab empleando librerías de análisis de datos y visualización.

## Objetivo
Determinar cuál de las cuatro tiendas presenta mejores condiciones de rendimiento considerando diferentes indicadores de ventas y satisfacción del cliente.

Para ello se analizan factores como:

- Ingresos totales por tienda
- Categorías de productos más vendidas
- Calificación promedio de clientes
- Productos más y menos vendidos
- Costo promedio de envío

## Contenido del repositorio

- `AluraStoreLatam.ipynb` — Notebook que contiene el análisis de datos, visualizaciones y conclusiones.

## Tecnologías utilizadas

- Python
- pandas
- matplotlib
- Jupyter Notebook / Google Colab
